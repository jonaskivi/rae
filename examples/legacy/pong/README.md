# Pong Example

A classic Pong game implemented in Rae using Raylib.

## Features
- Player-controlled paddle (W/S keys)
- Simple AI opponent
- Ball collision physics
- Score-less indefinite gameplay

## Best Practices Shown
- Functional decomposition (splitting logic into reusable functions)
- Constant-like helper functions for configuration
- Input handling and AI separation from main loop
- Using the `clamp` function for bounds safety
